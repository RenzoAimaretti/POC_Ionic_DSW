📋 Descripción del Proyecto
Este proyecto consiste en el desarrollo de un prototipo de aplicación To Do List como prueba de concepto (POC) para la cátedra de Desarrollo de Software. La aplicación permite a los usuarios gestionar tareas pendientes de forma sencilla y eficiente.

🛠 Tecnologías Utilizadas
Frontend:
Angular: versión 18
Ionic: versión 8
La elección de Angular e Ionic permite crear una experiencia de usuario fluida y responsive, compatible con múltiples dispositivos y sistemas operativos.

📌 Características Principales
Gestión de Tareas: Añadir, editar y eliminar tareas.
Interfaz Responsiva: Adaptación automática a dispositivos móviles gracias a Ionic.
Experiencia Intuitiva: Fluidez en la navegación y uso sencillo.

🚀 Instalación y Configuración
Prerrequisitos
Node.js: Versión 16 o superior.
Angular CLI: Versión 18
Ionic CLI: Versión 8
Instalación
* Clona el repositorio:

    ```bash
    git clone <URL_del_repositorio>
    ```

* Accede a la carpeta del proyecto:

    ```bash
    cd nombre-del-proyecto
    ```

* Instala las dependencias:

    ```bash
    npm install
    ```
Ejecución
Para iniciar el proyecto en modo de desarrollo, ejecuta:

 ```bash
    ionic serve
 ```

ionic serve
Esto abrirá la aplicación en tu navegador.

📂 Estructura del Proyecto
/src: Contiene el código fuente del proyecto.
/app: Componentes de Angular y lógica de la aplicación.
/assets: Archivos estáticos como imágenes y estilos.
/environments: Configuraciones de entornos (desarrollo y producción).
